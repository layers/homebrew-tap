# Layers Tap

## How do I install these formulae?

`brew install layers/tap/<formula>`

Or `brew tap layers/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "layers/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
